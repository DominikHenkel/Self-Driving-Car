# Self-Driving-Car

Die gewichte müssen unter https://pjreddie.com/media/files/yolov3.weights noch herrunter geladen werden. Wenn du sie entpackst bitte in /data/weights einfügen und natürlich die Pfade in den beiden .py Dateien anpassen. Anschließend musst du nur die test.py ausführen. Du brauchst nur Tensorflow und OpenCV als Dependencies.
